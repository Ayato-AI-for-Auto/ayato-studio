Nsight Computeのダウンロード後の初期セットアップ方法

Published: 01/09/2026 08:30:00

 

まず、

検索して開きます。

Nsight ComputeでGPUの解析を行うと、「レポート（分析結果）」が生成されます。その結果ファイルを開いたとき、「最初にどの画面を見せるか？」 という設定です。 NVIDIAは、詳細なデータがいきなり出るよりも、まずは**「Summary（全体の概要）」**を表示することを推奨しています。

 

↓

タイトル: 「NVIDIAソフトウェアの改善にご協力ください」

本文: NVIDIAのソフトウェアパッケージは、Google Analyticsを通じて以下の使用状況とプラットフォームデータを収集します：

収集データ 説明
Product version 使用しているNsight Computeのバージョン
Display device ディスプレイデバイス情報（GPU名など）
Session data セッション情報（使用時間、開いた機能など）
CPU information CPUの型番やスペック

同意した場合: このデータは匿名化され、NVIDIAソフトウェアの改善のみに使用されます。

選択肢:

  • 「Yes」: 使用状況の収集を有効化
  • 「No」: 使用状況の収集を無効化

補足:

  • この質問は一度だけ表示されます
  • 後から設定を変更したい場合は、 
    Tools > Options...
     メニューから変更可能

 

画面の解説

セクション 説明
Recent Files 最近使用したプロジェクト/ファイル（現在は空）
Quick Launch プロジェクトを作成せずに素早くプロファイリングを開始
New Project 新規プロジェクトを作成（設定を保存したい場合）
Load Existing 既存のプロジェクト/ファイルを読み込む

📢 重要な通知

画面下部に表示されている：

NVIDIA Nsight Compute 2025.4.0 is now available.

これをダウンロードすることで、CUDA 12.5 との互換性問題が解決する可能性が高いです！

 

ダウンロードする人は別の記事の「Nvidiaのアカウント構築からNsight Computeのダウンロードまで」の記事を参考にしてください。

 

 

↓

 

アプリケーションを設定

メニューバーから以下を選択：

  1. Connection → Connect...
  2. または上部ツールバーの ▷ Start Activity をクリック

3. Python アプリケーションを設定

表示されるダイアログで：

フィールド 値
Application
 
C:\Users\ayato\project\code\.venv\Scripts\python.exe
Working Directory 
C:\Users\ayato\project\code
Arguments 
gpu_test\main.py 起動したいスクリプトを選択

まず「▷ Start Activity」（左上のツールバー）をクリックしてみてください。

ダウンロードは完了。
